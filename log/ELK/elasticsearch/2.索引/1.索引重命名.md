elasticsearch 索引重命名 - 柒然的博客 - CSDN博客 https://blog.csdn.net/qq_34624315/article/details/83089794

直接复制索引到新的索引名称
POST localhost:9200/_reindex
{
  "source": {
    "index": "indexName"
  },
  "dest": {
    "index": "newIndexName"
  }
}

查询复制索引到新的索引名称
POST localhost:9200/_reindex
{
  "source": {
    "index": "indexName",
    "type": "typeName",
    "query": {
      "term": {
        "name": "shao"
      }
    }
  },
  "dest": {
    "index": "newIndexName"
  }
}
